![](https://github.com/krishpranav/BugHuntingToolKit/blob/master/Bug.gif)

Steps to run this tool:
  1. git clone https://github.com/krishpranav/BugHuntingToolKit
  2. cd BugHuntingToolKit
  3. sudo chmod 777 *
  4. python2 bugtoolkit.py
